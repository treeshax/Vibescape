# 🎧 Vibescape

## 📌 Description

A web application that uses the Spotify API to discover and explore music based on user moods.

---

## 🚀 Features (Planned)

🎵 Core Features
Fetch songs based on mood
Display song details (title, artist, album image)

🔍 Interactive Features
Search songs by name
Filter songs by artist
Sort songs (alphabetically / popularity)

🎨 UI Features
Responsive design
Clean layout with cards
Mood selection buttons

---

## 🔌 API

Spotify Web API
https://developer.spotify.com/documentation/web-api/

---

## 🛠️ Tech Stack

* HTML
* CSS (Tailwind)
* JavaScript (ES6)
* Fetch API
